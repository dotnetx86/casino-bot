## Описание продукта
- Telegram-бот на Python (aiogram).
- Пользователь может получать виртуальную валюту, играть в три мини‑игры, просматривать лидерборды.
- Админ команды для управления пользователями и балансами.
- Хранение: SQLite, Асинхронная обработка.

## Набор команд
- /start
- /help
- /deposit
- /withdraw
- /balance
- /leaderboard
- /admin_setbalance
- /admin_broadcast
- /roulette
- /mines
- /towers

## Описание работы команд

- /start  
    - Регистрирует пользователя, создаёт начальный баланс.

- /help  
    - Справка по командам.

- /random_text
    - Случайный текст из Торы.

- /deposit
    - Депозит Телеграм Звезд на баланс.

- /withdraw
    - Вывод баланса в Телеграм Звезды.

- /balance  
    - Показывает текущий баланс.

- /leaderboard  
    - Таблица лидеров по балансу.

- /admin_setbalance, /admin_broadcast  
    - Управление балансами и рассылками. Только для администраторов.

- /roulette
    - Бот запрашивает у пользователя ставку и цвет, на который он ставит.
    - Бот отсылает сообщение c линией разноцветных квадратов и стрелкой посередине.
    - Каждые 75 мс бот изменяет сообщение и линия сдвигается (постепенно интервал увеличивается).
    - Когда рулетка останавливается, если пользователь выиграл, то он получает вознаграждение. 

- /mines  
    - Бот запрашивает у пользователя ставку и количество мин. От количесва мин зависит множитель.
    - Бот отсылает сообщение с кнопками.  
    - Каждое открытие либо безопасно, либо встречается мина - проигрыш. Каждая открытая клетка увеличивает множитель.  

- /towers
    - Бот запрашивает у пользователя ставку и уровень сложности (чем сложнее тем больше множитель).
    - Бот отсылает сообщение со столбцами из кнопок.
    - Игрок поднимается по уровням, при каждом успешном открытии множитель растёт.
