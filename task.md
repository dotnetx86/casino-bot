## Цели / задачи

- Создать игровой Telegram-бот в котором есть рулетка, мины и башни.

## Описание продукта
- Telegram-бот на Python (aiogram).
- Пользователь может получать виртуальную валюту, играть в три мини‑игры, просматривать лидерборды.
- Админ команды для управления пользователями и балансами.
- Хранение: SQLite, Асинхронная обработка.

## Набор команд
- /start
- /help
- /profile
- /balance
- /leaderboard
- /admin_setbalance
- /admin_broadcast
- /roulette
- /mines
- /towers

## Описание работы команд

- /start  
    - Регистрирует пользователя, создаёт начальный баланс.

- /help  
    - Справка по командам.

- /profile  
    - Показывает имя, id, баланс и статистику.

- /balance  
    - Показывает текущий баланс.

- /leaderboard  
    - Таблица лидеров по балансу.

- /admin_setbalance, /admin_broadcast  
    - Управление балансами и рассылками. Только для администраторов.

- /roulette
    - Бот спрашивает у пользователя ставку и цвет, на который он ставит.
    - Бот отсылает сообщение c линией разноцветных квадратов и стрелкой посередине.
    - Каждые 200 мс бот изменяет сообщение и линия сдвигается (постепенно интервал уменьшается).
    - Когда рулетка останавливается, если пользователь выиграл, то он получает вознаграждение. 

- /mines  
    - Бот спрашивает у пользователя ставку и количество мин. От количесва мин зависит множитель.
    - Бот отсылает сообщение с кнопками, на которых изображен черный квадрат.  
    - Каждое открытие либо безопасно, либо встречается мина - проигрыш. Каждая открытая клетка увеличивает множитель.  

- /towers
    - Бот спрашивает у пользователя ставку и уровень сложности (чем сложнее тем больше множитель).
    - Бот отсылает сообщение со столбцами из черных квадратов.
    - Игрок поднимается по уровням, на каждом уровне шанс проиграть. При каждом успешном открытии множитель растёт.
